About the Project

Hands Off is an action/shooter created for GBCJAM #9 at George Brown College, where I studied Game Programming. The jam theme was "Uncomfortable," and the game commits fully to that: the mechanic is built around pushing away hands that are scratching a blackboard

Built by a two-person team in a single day. Playable directly in the browser via HTML5.

Technical Highlights

  • Completed in a single day by a two-person team
  • Browser-playable via HTML5

Features

  • Action/shooter gameplay with an unconventional hand-scratching premise

What I Learned

This was the first game jam experience at George Brown College. I learned than a simple, clearly scoped game can be built and shipped in one day if the design is disciplined from the start. Getting something complete and playable under that constraint is the whole goal.
