About the Project

VIZARD is a mystery game inspired by S.P.L.I.T. and The Rootrees are Dead. You play a hacker-for-hire tasked with exposing a mysterious company by connecting to employee workstations, reading private files, and piecing together the truth from internal chats and documents.

First built in one week for Global Game Jam 2026, the game is now in continued post-jam development. The central interface is a faithful Linux-style console the player uses to navigate the company's network. I served as both lead programmer and primary designer, writing the puzzles and dialogue while building the data-driven architecture that underlies the terminal.

Technical Highlights

  • Data-driven terminal system: console commands, file system, employee profiles, and chat logs are all defined in data - adding new content requires no code changes
  • Localization pipeline using Unity's table system connected to an external Google Sheet for content management
  • Ongoing post-jam development expanding the design and asset management systems

Features

  • Faithful simulation of a Linux-style console with many familiar commands
  • Awesome retro aesthetic in a modern world
  • Intriguing story that promises more than what initially seems

What I Learned

This was the first project where design ownership was the primary creative responsibility. The data-driven architecture meant the terminal could be extended without touching code, which proved valuable when scope grew post-jam. Working on localization via Unity's table and Google Sheet pipeline added a practical production skill that carries into larger projects.
