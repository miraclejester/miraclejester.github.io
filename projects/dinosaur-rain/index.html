About the Project

Dinosaur Rain is an action game built for GBCJAM #11, a game jam for Game Development students at George Brown College. The jam's theme was "Frantic," and the game delivers on it directly: dinosaurs rain from the sky and players must survive the onslaught.

Built in Unity by a three-person team in a single day, the result demonstrates solid scope judgment - a simple, focused premise that was achievable under a one-day constraint and communicates the theme without ambiguity. Playable in the browser or downloadable for Windows.

Technical Highlights

  • Jam theme "Frantic" directly expressed through high-density dinosaur spawning and survival pressure
  • Completed in a single day by a three-person student team
  • Browser-playable via HTML5 and downloadable Windows build

Features

  • Frantic action gameplay centered on a dinosaur invasion theme
  • Browser-playable via HTML5
  • Downloadable Windows build

What I Learned

A one-day jam demands a design that can be finished in one day. The main skill on display here is scope judgment: identifying a premise simple enough to build completely under the constraint, while still being coherent and fun. The team hit that target, which is the whole point of a short jam.
