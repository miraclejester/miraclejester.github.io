About the Project

Chicken Feud is a local multiplayer puzzle game made for the Venezuela Duel Jam. It was heavily inspired by Tetris Attack on the SNES. Sir Areponcio is on his way to rescue the princess, but an evil sorcerer keeps placing irresistible roasted chicken in his path. One player is the princess, pulling the hero forward. The other is the sorcerer, derailing him with temptation.

Two players, same machine, presented in Spanish. The technical challenge was creating a puzzle interface that worked at any screen resolution, with competing player goals legible simultaneously on one display. Playable in the browser.

Technical Highlights

  • Local multiplayer: one player pursues rescue, the other plays saboteur
  • Puzzle interface built to scale cleanly across screen resolutions
  • Browser-playable via HTML5

Features

  • Design inspired by Tetris Attack
  • Local two-player competitive puzzle gameplay on the same machine
  • Quirky chicken-themed narrative and humor

What I Learned

The design was heavily inspired by Tetris Attack, which meant studying how that kind of grid-based interface is constructed before building it. The most technically satisfying outcome was a resolution-independent puzzle layout which is a non-trivial UI engineering problem when the grid has to feel tight and precise.
