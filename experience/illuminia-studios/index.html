About the Role

At Illuminia Studios, Jose was the Gameplay Programmer for BOAA, a pick-up-and-play arcade game with an Afrofuturist aesthetic, released on Android, iOS, and the Microsoft Store. Version 0.4.0 included arcade mode, multiplayer, and leaderboards.

The studio was working toward a story mode and EOS blockchain integration for player collectibles before ceasing operations due to funding constraints. BOAA built a small dedicated community on the Microsoft Store before the shutdown.

Key Responsibilities

  • Character controller: built the core player movement and feel for an arcade action game
  • Enemy design and implementation
  • Playing field hazards system
  • UI implementation across game screens
  • Shipped to three platforms: Android, iOS, and Microsoft Store

What I Learned

Illuminia was the first startup experience with fast-paced development, direct trade-offs against deadlines, and the reality that a technically solid product can still be halted by funding rather than quality. It reinforced existing Unity expertise and introduced the practical discipline of shipping to multiple mobile storefronts simultaneously.
