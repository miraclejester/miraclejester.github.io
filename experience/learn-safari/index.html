About the Role

At Learn Safari, I was a Gameplay Programmer on Spanish Safari: the company's primary product teaching Spanish to children aged 3 to 9 through a mix of 3D world exploration and 2D educational minigames. I was involved from early development through the 1.0 release in October 2016.

Built in Unity for iOS and web, the game uses a cartoony 3D overworld connecting to self-contained 2D minigames that reinforce language lessons. This was my first professional role in game development.

Key Responsibilities

  • 3D navigation systems for player characters in the overworld
  • Internal inventory management system
  • Minigame development end-to-end: design, implementation, and integration with the main game loop
  • Multiple distinct 2D minigames, each targeting a specific language learning objective
  • Shipped to iOS and web as v1.0

Features

  • Fun minigames that make it easy to learn
  • Cute world with friendly characters
  • Lessons are still being added to the game
  • 3D navigation systems for the characters

What I Learned

Learn Safari was my first professional role in game development. It established the habits that I carry through all later work: estimating deliverables, collaborating within a team under production constraints, and building the kind of Unity debugging and optimization fluency that only comes from shipping real software. The minigame architecture of self-contained modules that interface cleanly with the main game became an early model for thinking about system boundaries.
